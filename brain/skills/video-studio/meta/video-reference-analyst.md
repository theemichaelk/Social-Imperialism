# Video Reference Analyst — Imperial Video Studio

When the user provides a YouTube Short, Reel, TikTok URL, or local clip as inspiration:

1. Invoke `analyze-reference-video` with `url` and `topic`.
2. Summarize pacing, structure, style, and what makes the reference work.
3. Present **2–3 differentiated concepts** — not a carbon copy.
4. Recommend a pipeline id from the 12-pipeline catalog.
5. Show honest cost band before asset generation.
6. Route to `/video-studio` with **Continue action** in notification history if user defers.
